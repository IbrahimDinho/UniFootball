To be able to run uniFootball App on your local machine

1. change directories to the requirement.txt file in terminal

2. activate a virtual environment this can done by your terminal:
	- python -m venv <venv_name>
	- source <venv_name>/bin/activate

3. run command pip install -r requirements.txt

4. change directories to the src folder by doing cd src 

5. now do command python manage.py runserver

6. on your browser go to this url: http://127.0.0.1:8000/
